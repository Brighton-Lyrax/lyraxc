# Lyraxc

Autonomous browser agent that performs human-like actions in a real browser from plain-language instructions.

## Features

- Browser automation using Puppeteer/Playwright
- Plain-language instruction parsing
- Human-like interaction patterns
- Screenshot and DOM analysis
- Multi-step task execution
- Error recovery and retry logic

## Tech Stack

- **Language**: TypeScript/JavaScript
- **Automation**: Puppeteer / Playwright
- **NLP**: OpenAI GPT / Anthropic Claude for instruction understanding

## Getting Started

```bash
git clone https://github.com/Brighton-Lyrax/lyraxc.git
cd lyraxc
npm install
npm run dev
```
